This app is not code-signed because it's free and open-source. We don't pay Apple's $99/year Developer Program fee to keep this completely free for you.

How to Open on macOS:

  1. Download the DMG file from the link above
  2. Open the DMG and drag RSS Reader to your Applications folder
  3. First Launch: You'll see a warning that the app "cannot be opened because it is from an unidentified developer"
  4. Bypass the warning:
    • Go to System Settings (or System Preferences) ‚Üí Privacy & Security
    • Scroll down to find the message about RSSReader being blocked
    • Click "Open Anyway"
    • Confirm by clicking "Open" in the dialog

Alternative Method (Right-Click):

  1. In Finder, right-click (or Control-click) the RSS Reader app
  2. Select "Open" from the menu
  3. Click "Open" in the confirmation dialog

Is this safe? Yes! The source code is fully open and available on GitHub. The warning appears only because we haven't paid for Apple's Developer Program, not because there's anything wrong with the app. You only need to do this once - subsequent launches will work normally.
